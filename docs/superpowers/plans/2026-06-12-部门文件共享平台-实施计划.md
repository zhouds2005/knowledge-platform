# 部门文件共享平台 — 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 从零搭建独立的部门文件共享平台，包含文档管理、Wiki 协同、Nextcloud 网盘集成、文档级 RBAC、空间审核流、知识门户、知识图谱。

**Architecture:** 独立 React SPA 前端 + Express API 后端 + PostgreSQL 数据库，统一知识对象模型贯通文档/Wiki/网盘三类实体，权限判定和审核流在对象层统一处理。

**Tech Stack:** React 18 + Vite + TypeScript + Tailwind + shadcn/ui + Drizzle ORM + PostgreSQL + Express + Nextcloud API

**Target DB:** `knowledge_platform` (独立数据库)

**Target Directory:** `D:\企业档案管理系统1.0\knowledge-platform\`
